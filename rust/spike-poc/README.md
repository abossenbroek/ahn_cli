# `ahn-hf-spike` ‚Äî THROWAWAY interop spike (hf-v2 Task 1b)

This is **not** the shippable `ahn-heightfield` crate. It exists only to
falsify or rule out the v2 exchange-format interop doubts on
`{ubuntu, macos, windows}` before implementation begins, and it is
deleted/superseded in Phase 4.

It contains a minimal Rust decoder for:

- the **v2 `.hf` chunk** (120-byte header + CRC + pad + checksum-on zstd frame), and
- the **`AHNP` pack container** (128-B header, level directory, 96-B index
  entries, SHA-256 hash section, aligned blob region),

mirroring the throwaway Python producer POC exactly. The tests read committed
golden vectors and pack fixtures from `tests/data/` (regenerated by
`tools/gen_vectors.py`) plus the repo's loose `.hf`/`.jpg`/`.glb` fixtures under
`tests/tiles3d/fixtures/rust-consumer/`.

## Reproduce

```bash
# regenerate the golden vectors + pack fixtures (byte-deterministic)
uv run python rust/spike-poc/tools/gen_vectors.py
# run the spike
cd rust/spike-poc && cargo test --all-features
```

Findings and the doubt-by-doubt table live in
`docs/superpowers/specs/2026-07-12-hf-v2-spike-results.md`.
